Privacy Policy

Hey tech squad, we know how much privacy matters to you so this is what this page is ment for.

  • Data gathering
    • Yes this website does gather basic data in order to enhance the overall experience
    • Along with all posts, usernames as well as dates are stored
    • The quizes and a user's responses are not stored and merely serve as a means to reward the user if his/her answer is correct
    • The themes and color forms used are also associated with each independent user to provide a more personal experience and nothing more. The themes and colours are for users to change the website to their own personal taste.
    • Each users points are also managed
    • No location based data is collected
    • This website does not track IP addresses of any of its user
    • Sensitive information such as passwords are encrypted
  • Uploads
    • Any uploaded material such as text on the message board is stored on the database
  • Access
    • If he/she feels the need to see their data, contact one of the execs and this can be arrange
    • Though users cannot be granted access into the database, they request for a file to view all the data accumulated on the website pertaining to them and them alone.
    • Users that request for the information of others will be denied
  • Records
    • Records within tables are kept update to date as users make changes on the website
    • To maintain accurate records most messages have a date stamp
  • Purpose
    • The point of the data we collect is to improve your own experience with our service
    • None of the data on the website is sold or disclosed to any third-party
    • It is used purely in the scripts of the website and for analysis
    • We monitor use of certain features to see if there are useful at all
    • Admins are not at liberty to mess with another users data
    • Sensitive info is secured, as security is a high priority for us

If you guys have any further questions or concerns please do feel free to drop us an email using the address that can be found in the footer
